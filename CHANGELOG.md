# 更新日志

所有重要的更新都会记录在此文件中。

## v3.3.1

### 优化

- **音频引擎简化**：移除了 FFmpeg 和 MPV 音频处理引擎，默认使用 Web Audio 引擎，避免切换引擎导致的启动报错问题
- **代码精简**：删除了与 FFmpeg/MPV 相关的代码，减少维护负担
- **性能优化**：优化了内存占用，包括：
  - 减小 Blob URL 缓存限制（50→15）
  - 添加歌词预加载缓存限制（最多 5 首）
  - 精简音乐状态持久化字段
  - 减少播放历史记录限制（500→200）

### 修复

- **启动报错**：修复了切换音频引擎后重启应用导致的 `.node` 文件锁定问题

---

## v3.3.0

### 新功能

- **歌词切换菜单**：在底部播放栏添加了歌词切换菜单，鼠标悬停即可选择开启桌面歌词或任务栏歌词

### 优化

- **xeapi 初始化优化**：添加了缓存检查和重试机制，确保网络不稳定时也能正常获取加密公钥
- **API 初始化顺序优化**：调整了初始化顺序，先获取 xeapi 公钥，再注册匿名用户，避免首次请求失败
- **日志输出优化**：精简了错误日志，避免输出冗长的 axios 配置信息

### 修复

- **网易云音乐兼容性**：修复了因 CommonJS 模块导入方式导致的启动报错问题
- **首次加载歌单失败**：修复了首次打开歌单详情时可能出现的"获取失败"问题

---

## v3.2.0

### 新功能

- **抖音收藏音乐播放**：支持获取并播放抖音收藏的音乐
- **一键播放全部**：支持一键播放抖音收藏列表中的所有音乐
- **Cookie 文件管理**：支持从文件加载抖音 Cookie，自动复制到项目目录
- **自动批量加载**：支持自动批量加载抖音收藏列表，无需手动点击加载更多

### 技术重构

- **抖音 API 重写**：使用 TypeScript 重写了 SM3 哈希和 ABogus 签名算法，不再依赖 TikTokDownloader 项目
- **原生模块构建**：集成了 Rust 原生模块用于音乐分析和任务栏歌词

### 其他

- 项目更名为 **KfunMusic**
- 更新许可证为 GPL-3.0
- 更新 GitHub 仓库链接为 `Lrht-llw/KfunMusic`
- 删除 macOS 和 Linux 相关配置，仅支持 Windows
